# ONLYDROPS

Each song on **ONLYDROPS** is limited to **30 seconds** to keep up the energy. 
Get started by uploading your songs and discover hundrets of playlists!

## Tech stack

- Angular
- Angularfire
- Firebase
- Tone.js

## Build instructions

*checkout master branch*

```npm install```

```ng serve```
